# GPRADFWI.jl Docs

This directory contains the dedicated Julia package documentation for `GPRADFWI.jl`, built with Documenter.jl.

## Build Locally

From the package root:

```bash
julia --project=docs docs/make.jl
```

## Docs Layout

- `docs/Project.toml`: documentation environment
- `docs/make.jl`: Documenter build script
- `docs/src/index.md`: overview and package structure
- `docs/src/examples.md`: package workflows and reproducibility entry points
- `docs/src/api.md`: exported API reference
